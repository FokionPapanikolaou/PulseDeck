# PulseBar — Καταχώριση Store (Ελληνικά)

## Όνομα εφαρμογής
PulseBar

## Υπότιτλος / Σύντομη περιγραφή (μέχρι 200 χαρακτήρες)
Ο σφυγμός του PC σου — πάνω στην μπάρα των Windows 11. Live CPU, RAM, GPU, δίκτυο και καιρός, πάντα σε μια ματιά.

## Περιγραφή (μέχρι 10.000 χαρακτήρες)
Το **PulseBar** μετατρέπει την μπάρα των Windows σε ζωντανό πίνακα ελέγχου. Μικρό, διάφανο, πάντα ορατό — αλλά ποτέ ενοχλητικό.

Δες **CPU**, **RAM**, **GPU**, ταχύτητα **δικτύου** και τον **καιρό** με μια ματιά, χωρίς να ανοίγεις το Task Manager. Πέρασε το ποντίκι πάνω σε μια μέτρηση για άμεσες λεπτομέρειες: κορυφαίες διεργασίες, φορτίο ανά πυρήνα, VRAM σε χρήση / σύνολο, σύνολα συνεδρίας δικτύου, I/O ανά δίσκο.

### ✨ Τι δείχνει
• **CPU** — χρήση % και ζωντανή ταχύτητα ρολογιού σε GHz
• **RAM** — σε χρήση / σύνολο GB, καθώς και % φορτίο
• **GPU** — χρήση % και VRAM σε χρήση / σύνολο *(μέσω μετρητών Windows — χωρίς επιπλέον εργαλεία)*
• **Δίκτυο** — upload ▲ / download ▼ σε πραγματικό χρόνο (MB/s ή Mbps)
• **Δίσκος** — ταχύτητα ανάγνωσης/εγγραφής και χώρος % ανά δίσκο
• **Μπαταρία** — % με ένδειξη φόρτισης (φορητοί)
• **Καιρός** — τρέχουσα θερμοκρασία, με 3ήμερη πρόγνωση στο hover

### 👁️ Hover για λεπτομέρειες
Σταμάτα το ποντίκι σε κάθε μέτρηση για ένα καθαρό popup:
• CPU → κορυφαίες διεργασίες + μπάρες ανά πυρήνα
• RAM → οι μεγαλύτεροι «καταναλωτές» μνήμης
• GPU → VRAM σε χρήση / σύνολο
• Δίκτυο → σύνολα συνεδρίας & συνολικά
• Δίσκος → ανάγνωση/εγγραφή ανά φυσικό δίσκο
• Καιρός → αίσθηση, υγρασία, άνεμος, 3ήμερη πρόγνωση

### 🎨 Φτιάξ' το όπως σ' αρέσει
• Κάθεται **ΠΑΝΩ** στην μπάρα (δίπλα στα icons) ή **ακριβώς από πάνω** — με έναν διακόπτη
• Σύρσιμο όπου θες, κλείδωμα θέσης
• **10 χρωματικά θέματα** συμπεριλαμβανομένων gaming (Neon, Cyber, Inferno, Plasma, RGB με animation)
• 3 μεγέθη, ρυθμιζόμενη διαφάνεια & συχνότητα ανανέωσης
• Mini sparkline γραφήματα για CPU & RAM

### ⚡ Σχεδιασμένο να εξαφανίζεται
• Κρύβεται μαζί με την μπάρα σε fullscreen games/βίντεο — επιστρέφει στο Alt-Tab, όπως το ρολόι
• Μία μόνη παρουσία, ελάχιστο αποτύπωμα (~15 MB RAM)
• Προαιρετική εκκίνηση με τα Windows
• Οι ρυθμίσεις είναι στο tray icon — το widget παραμένει καθαρό

### 💜 Δωρεάν, για πάντα
Το PulseBar είναι **δωρεάν**. Αν σου αρέσει, υπάρχει σύνδεσμος Donate στο tray menu (PayPal / Revolut).
