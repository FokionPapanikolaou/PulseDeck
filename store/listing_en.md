## Description (max 10,000 chars)
**PulseBar** turns your Windows taskbar into a live system dashboard. Tiny, transparent, always visible — but never in the way.

See your **CPU**, **RAM**, **GPU**, **Network** speed and even the **weather** at a glance, without ever opening Task Manager. Hover any metric for instant details: top processes, per-core load, VRAM used / total, network session totals, per-disk I/O.

### ✨ What it shows
• **CPU** — usage % and live clock speed in GHz
• **RAM** — used / total GB, plus % load
• **GPU** — usage % and VRAM used / total *(via Windows performance counters — no extra tools)*
• **Network** — real-time upload ▲ / download ▼ (MB/s or Mbps)
• **Disk** — read / write speed and per-drive space %
• **Battery** — % with charging indicator (laptops)
• **Weather** — current temperature, with a 3-day forecast on hover

### 👁️ Hover for details
Point at any metric for a clean popup panel:
• CPU → top processes + per-core bars
• RAM → top memory consumers
• GPU → VRAM used / total
• Network → session and lifetime totals
• Disk → read / write per physical disk
• Weather → feels-like, humidity, wind, 3-day forecast

### 🎨 Make it yours
• Sit **on** the taskbar (next to your app icons) or float **just above** it — one toggle
• Drag it anywhere; lock its position to avoid accidents
• **10 color themes** including gaming styles (Neon, Cyber, Inferno, Plasma, animated RGB)
• 3 sizes, adjustable transparency and refresh rate
• Mini sparkline graphs for CPU & RAM

### 🌍 8 languages
English · Ελληνικά · Español · Deutsch · Français · Italiano · Português · Русский
*Auto-detects your Windows display language.*

### ⚡ Designed to disappear
• Hides automatically with the taskbar in fullscreen games and videos — reappears on Alt-Tab, just like the clock
• Single instance, lightweight (~15 MB RAM, tiny CPU)
• Optional start with Windows
• Settings live in the system-tray icon — the widget itself stays a clean overlay

### 💜 Free, forever
PulseBar is **free**. If you enjoy it and want to support development, a Donate link lives in the tray menu (PayPal / Revolut).
